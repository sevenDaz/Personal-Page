# 家庭烘焙经验分享

## 开始前的准备

家庭烘焙是一件有趣且富有成就感的事情。以下是我的一些经验分享。

## 基础工具

- 烤箱
- 烘焙模具
- 打蛋器
- 电子秤
- 搅拌碗

## 食材选择

- 高筋面粉：适合面包制作
- 低筋面粉：适合蛋糕和饼干
- 新鲜鸡蛋
- 优质黄油

## 烘焙技巧

### 1. 面团发酵

```text
- 温度控制在28-30°C
- 发酵时间根据室温调整
- 观察面团大小变化
```

### 2. 烤箱温度

- 预热烤箱至适当温度
- 使用烤箱温度计确保准确
- 不同食材需要不同温度

## 经典配方

### 巧克力曲奇

```text
材料：
- 黄油 100g
- 糖 70g
- 鸡蛋 1个
- 面粉 200g
- 可可粉 20g
- 巧克力碎 100g

步骤：
1. 黄油软化后加糖打发
2. 加入鸡蛋搅拌均匀
3. 筛入面粉和可可粉
4. 加入巧克力碎
5. 烤箱180°C烤15分钟
```

## 常见问题解决

### 面包塌陷

- 发酵过度
- 烤箱温度不够
- 面团水分过多

### 蛋糕开裂

- 烤箱温度过高
- 搅拌过度
- 烘烤时间不足

## 小贴士

- 记录每次烘焙的配方和结果
- 逐步尝试新配方
- 享受烘焙的过程

## 总结

家庭烘焙需要耐心和练习，但每次成功都会带来巨大的满足感。
